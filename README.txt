數據管理
========

自動恢復（首次訪問）
------------------
系統首次加載時會自動：
1. 檢查 LocalStorage 是否為空
2. 從 localstorage_backup.json 恢復數據
3. 加載所有系統功能
4. 顯示在瀏覽器控制台中的日誌

導出備份
--------
定期導出您的數據：
1. 點擊右上角「💾 匯出數據」
2. 文件自動下載
3. 文件名：ys89_backup_YYYY-MM-DD.json
4. 可以保存多個版本

數據統計
--------
查看數據完整性：
1. 點擊右上角「📈 數據統計」
2. 在瀏覽器控制台（F12）查看統計表
3. 檢查各集合的項數

常見問題
========

Q: 數據會丟失嗎？
A: 不會。數據保存在瀏覽器的 LocalStorage 中。只有在清除瀏覽
   器數據或卸載擴展程序時才會丟失。建議定期導出備份。

Q: 如何備份我的數據？
A: 點擊右上角「💾 匯出數據」按鈕，系統會自動下載一個包含所有
   數據的 JSON 文件。

Q: 如何恢復舊備份？
A: 請參考 USAGE_INSTRUCTIONS.txt 文件中的「恢復導出的舊備份」
   部分。

Q: 可以在多台電腦上使用嗎？
A: 可以。每台電腦都有獨立的 LocalStorage。您可以通過導出和匯
   入備份文件在不同電腦間同步數據。

Q: 如何清除所有數據？
A: 在瀏覽器控制台（F12）執行 clearAllData() 命令。系統會提示
   確認。

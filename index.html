💡 怎麼運作（完整流程）
①
在這裡 選水軍 → 選平台 → 選落地頁，系統自動產生帶 utm_source / utm_medium / utm_content 的追蹤連結。水軍名單來自 社群水軍號，那邊改 utm 碼這裡即時同步。
②
填 ⑥標題（這條給哪篇用）、⑦用途（文章發布／炒話題），把連結貼到 lihi 縮短後，短連結貼回⑦下方欄。
③
選 ⑦ 發文分類（社群平台炒群／匿名社群發文／臉書社團粉專），按 💾 儲存 → 同時存進短連結庫 並自動到對應分頁建立一筆發文記錄（標題・UTM碼・短連結・時間）。
④
之後到 發文追蹤 / 炒話題追蹤 記錄發文時，用「📎 帶入已建的 UTM/短連結」下拉，一鍵帶入 UTM碼＋短連結＋標題，不用重 key。
⑤
系統每天把這些碼寫進對外 social-feed.json，SEO 成效儀表板用 utm_content 接 GA4，自動算出「這篇貼文帶幾人進論壇」。GA4 分頁也會用水軍中文名顯示 utm_source。

📖 使用說明

一頁看懂：角色怎麼分 · 發文記在哪 · 怎麼掛碼 · 成效怎麼串到 GA4

① 角色怎麼分（🎭 角色主檔）

所有角色集中在「角色主檔」，用 UTM 碼去重（同一人只一筆），分四類：

  • 🎯 專業導論壇主力：排雷週記/財神/阿奇/蟹蟹/kk — 導流論壇文章
  • 💬 社群水軍：IG/Threads/FB 真人角色，炒群呼應
  • 📘 FB社團角色：阿埔、陳玉樺等
  • 📝 鋪文/匿名平台：Medium/Pixnet/方格子/Dcard 等第三方帳號

每個角色可勾「能跑哪些管道」；左側「類型」下拉可重新歸類。

② 發文記在哪（三個發文分頁）
分頁放什麼
🔥 社群平台炒群IG/Threads/FB 炒話題、炒熱度
📝 匿名社群發文Medium/Pixnet/方格子/Dcard/PTT 等第三方部落格・論壇
📘 臉書社團/粉專FB 社團經營、粉專貼文

每個分頁都能直接「+ 新增記錄」，也能用平台篩選只看某平台、追蹤存活/互動。

③ 怎麼發文＋掛碼（🏷️ UTM 產生器＝中央入口・推薦）

發任何要追蹤的文，統一從 UTM 產生器走，一次搞定產碼＋短連結＋建記錄：

  1. ① 選角色（依類型分組）→ ② 選貼在哪 → ③ 活動 → ④ 落地頁 → ⑤ 自動產貼文編號
  2. ⑥ 填文章/貼文標題（摘要）
  3. ⑦ 選發文分類：🔥社群平台炒群／📝匿名社群發文／📘臉書社團粉專
  4. 把產生的連結縮短，短連結貼回
  5. 按 💾 儲存 → 自動存進短連結庫，並到對應分頁建立一筆發文記錄（狀態「已發」）
💡 例：發 Dcard → 選蟹蟹/陳坤、⑦選「匿名社群發文」、平台填 Dcard、填標題、貼短連結、按儲存 → 匿名社群發文分頁立刻出現這筆。
④ 掛碼規格（utm_content ＝ 對接主鍵）

每篇唯一，格式 角色碼-月日-當日序號（例 akki-0611-1），序號跨所有分頁自動遞增不撞號。

關鍵人工步驟：產生的 utm_content 要真的放進你貼出去的短連結裡（短連結 redirect 到 ys89 落地頁時帶這個碼），GA4 才收得到。

⑤ 成效怎麼串到 GA4

系統每天自動把三個分頁有掛碼的記錄寫進對外 social-feed.json → SEO 成效儀表板用 utm_content 接 GA4，自動算出「這篇貼文 → 帶幾人進論壇」。你只要好好掛碼、發文時記錄，數字會自己長出來。

bio / post / story / comment / desc / ad / dm
目的代碼
live / forum / lineys / linesport
register / promo / schedule / taiwantime
104matches / risk / freewatch
注意事項
  1. 未登記的短連結不得上線。
  2. SEO頁面不使用短連結。
  3. 主帳固定入口需依平台分開產生短連結。
  4. 角色帳 Bio 一個平台一條短連結。
  5. 廣告連結與自然貼文連結不可共用。
  6. 貼文短連結只用於重點貼文，不是每篇都產。
  7. 所有短連結名稱必須符合 yswc_帳號_平台_位置_目的 格式。

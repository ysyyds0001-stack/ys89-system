YS89 發文管理系統 - 快速參考卡
==============================

文件清單
========

index.html (65KB)
- 主程序文件
- 包含數據恢復函數
- 包含導出/統計功能
- 所有系統功能保留

localstorage_backup.json (32KB)
- 完整備份數據
- 12 個 LocalStorage 集合
- 176+ 個數據項

文檔文件
- DATA_RECOVERY_GUIDE.txt - 完整恢復指南
- USAGE_INSTRUCTIONS.txt - 詳細使用說明
- RECOVERY_SUMMARY.txt - 項目總結報告
- QUICK_REFERENCE.txt - 本快速參考卡

快速操作
========

首次訪問：
→ 打開 index.html
→ 自動恢復所有數據
→ 系統立即可用

導出備份：
→ 點擊導航欄「💾 匯出數據」
→ 文件自動下載
→ 文件名格式：ys89_backup_YYYY-MM-DD.json

查看統計：
→ 點擊導航欄「📈 數據統計」
→ 控制台 (F12) 顯示統計表

手動恢復（控制台）：
→ 打開 F12 進入控制台
→ 執行：recoverLocalStorageData()
→ 等待恢復完成

查看所有數據（控制台）：
→ 打開 F12 進入控制台
→ 執行：getDataStats()
→ 查看表格統計

核心函數
========

recoverLocalStorageData()
- 作用：自動恢復備份數據
- 返回：true/false
- 調用：自動（頁面加載時）

exportLocalStorageData()
- 作用：導出當前 LocalStorage
- 返回：無（自動下載）
- 調用：按鈕或 exportLocalStorageData()

getDataStats()
- 作用：顯示數據統計
- 返回：統計對象
- 調用：按鈕或 getDataStats()

clearAllData()
- 作用：清除所有數據
- 返回：無（需確認）
- 調用：clearAllData()

localStorage API 使用
====================

查看文章：
JSON.parse(localStorage.getItem('ys89_articles'))

查看角色：
JSON.parse(localStorage.getItem('ys89_personas'))

查看帳號：
JSON.parse(localStorage.getItem('ys89_community'))

查看發文：
JSON.parse(localStorage.getItem('ys89_posts'))

查看所有鍵：
Object.keys(localStorage).filter(k=>k.startsWith('ys89_'))

數據結構概覽
============

ys89_articles - Array
- id: 文章 ID
- title: 文章標題
- dir: 發布方向
- postedBy: [{ pid, name, date }]

ys89_personas - Array
- id: 角色 ID
- name: 角色名稱
- followers: 粉絲數
- tone: 語氣風格
- postPrompts: 文案記錄

ys89_posts - Array
- id: 發文 ID
- date: 發文日期
- personaName: 發文人角色
- platform: IG/FB/IG+FB
- likes, comments, shares, reach

ys89_community - Array
- id: 帳號 ID
- name: 帳號名稱
- direction: 回覆方向
- status: 活躍/人設建立中/暫停

ys89_routing - Object
- currentPage: 當前頁面
- lastVisited: 最後訪問時間
